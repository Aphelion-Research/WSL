# Collaboration

SSH endpoint:

```bash
ssh Martin@100.95.35.80
```

tmux sessions:

```bash
tmux attach -t matin
tmux attach -t dan
tmux new -A -s dominion
```

Scrolling in tmux: press `Ctrl-b`, then `[`, use arrows/PageUp/PageDown, press `q` to exit copy mode.

Remote file editing should use VS Code Remote SSH. Plain SSH cannot open Dan's local Explorer.

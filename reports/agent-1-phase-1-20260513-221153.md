## Bugs Found and Fixed

| Bug | Location | Fix |
|-----|---------|-----|
| `_BUILTIN_PATH_DENY` patterns required leading `/`, failing on relative test paths | `ignore.py` | Changed to `(?:^|[/\\])` prefix |
| `/tmp/` pattern blocked pytest's `tmp_path` fixture | `ignore.py` | Changed to `[^/\\][/\\]tmp[/\\]` — only blocks subdirectory `tmp/` |
| `_COMMENT_RE` used `re.DOTALL` with `.*$` causing single-line comments to consume entire file | `semantic_diff.py` | Changed to `[^\n]*` for single-line patterns |
| `_percentile()` test expectations used n-1 index logic (numpy-style) vs implementation's index-based | `test_bench.py` | Fixed test expectations |
| `pytest.ini` didn't include `dominion_loader/tests/` in `testpaths` | `pytest.ini` | Added `dominion_loader/tests` to testpaths |

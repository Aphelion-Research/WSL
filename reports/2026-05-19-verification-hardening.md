## Remaining Risks / Deferred Work

1. **`compile_source.py` does not filter `EXCLUDE_NAMES` subdirs at walk time.**
   The `compileall.compile_dir` call respects the filesystem but does not
   explicitly prune `__pycache__/` subdirs inside source roots.  This is safe
   because `.pyc` files are in `__pycache__/` and `compileall` skips them;
   however, if a source root ever acquires a large vendored sub-tree, explicit
   pruning should be added.

2. **`ragd_vault` package** was not confirmed to exist on disk during this run
   (it was silently skipped as "missing optional package").  If the package is
   intended to be present, confirm its path and add a presence assertion.

3. **pytest `2 deselected`** are integration-marked tests.  They are
   intentionally skipped by the default `-m "not integration"` filter.
   They should be run separately against a live environment periodically.

4. **`data/` partial gitignore** — committed dataset fixtures remain tracked.
   If the dataset grows, consider moving fixtures to a dedicated `fixtures/`
   directory and adding `data/` as a full gitignore entry with explicit
   negations for fixtures.
